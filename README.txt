# FinanceCot — Sitio estático listo para publicar

Páginas HTML con Tailwind vía CDN (no requiere build).

## Publicación gratis
### GitHub Pages
1. Crea un repo y sube estos archivos a `main`.
2. Settings → Pages → Deploy from a branch (`main` / root).
3. Reemplaza https://financecot.example en `sitemap.xml` y `robots.txt` por tu URL real.

### Netlify
1. Add new site → Deploy manually.
2. Arrastra la carpeta o el ZIP.

## Personalización
- Edita `index.html` (textos, enlaces de afiliado).
- Ajusta páginas legales a tu normativa.
